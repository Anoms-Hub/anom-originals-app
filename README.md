# ✦ Anom Originals App

**AI-powered social branding tool built for the fubar.com community.**

> Turn your personality into art. Digital identity meets e-commerce.

---

## 🛠 Tech Stack

| Layer | Tech |
|-------|------|
| **Frontend** | TypeScript, Vite, React |
| **Backend** | TypeScript (Express) |
| **Database** | Drizzle ORM (PostgreSQL) |
| **Testing** | Vitest |
| **Payments** | Cash App integration |
| **Fulfillment** | Printful API |
